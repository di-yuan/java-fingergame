# java-fingergame

## 簡介
1. 遊戲結合聊天室：邊玩遊戲還能邊與對方暢聊
2. 遊戲分3種 (1) battle：連線手指遊戲 (2) piano：鋼琴消消樂 (3) poking：舒壓戳手指遊戲
